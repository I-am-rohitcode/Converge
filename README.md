# Converge 💬

A real-time chat application built using Android and Firebase.

## 🚀 Features
- User Authentication (OTP / Firebase)
- Real-time Messaging
- Online / Offline Status
- Clean UI Design

## 🛠 Tech Stack
- Java (Android)
- Firebase Authentication
- Firebase Realtime Database / Firestore

## 📱 Setup
1. Clone the repository
2. Open in Android Studio
3. Add your Firebase config file
4. Run the app
